## ✨ Features

* ✅ Smooth scroll & entrance animations (Framer Motion)
* ✅ Fully responsive & mobile-optimized layout
* ✅ Clean product showcase
* ✅ Working contact form (via EmailJS)
* ✅ Fast performance via Vite & Vercel

## 🛠️ Tech Stack

| Layer             | Tech Used     |
| ----------------- | ------------- |
| **Frontend**      | React, Vite   |
| **Styling**       | Tailwind CSS  |
| **Animations**    | Framer Motion |
| **Form Handling** | EmailJS       |
| **Hosting**       | Vercel        |

## ⚙️ Run Locally

```bash
git clone https://github.com/sharad/jmo-exports.git
cd jmo-exports
npm install
npm run dev
```

Visit `http://localhost:5173` in your browser.
